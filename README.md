# Kunskapskontroll
Project Description

This repository contains my solution for Kunskapskontroll 2 and Kunskapskontroll 3 in the course Python Programming and AI Strategy.

### Dataset

The dataset used in this project is called housing.csv and cars_data.csv used for Kunskapskontroll 3  and Kunskapskontroll 2 respectivly.

### How to Run the Analysis

Open the Jupyter Notebook file

Run the cells from top to bottom

All results and visualizations are generated automatically
